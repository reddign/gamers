API Endpoints
-------------

Admins
==========

http://localhost/data_src/api/admin/read.php
http://localhost/data_src/api/admin/create.php



Hangman
==========

http://localhost/data_src/api/hangman/list.php
http://localhost/data_src/api/hangman/word.php
http://localhost/data_src/api/hangman/create.php
http://localhost/data_src/api/hangman/delete.php

Under the data_src folder we are using a REST API created using Core PHP. All CRUD operations on the database will be done using this code.

While you're trying to Create, Update, or Delete through API, make sure your provided data are in JSON format because header only accepts Content-Type: application/json in these requests.

Admins

These are accounts that are able to make changes to the data from within settings.

Get Admin

To get an admin you need to make a GET call to the following url :
http://localhost/data_src/api/admin/read.php?username=admin_username 


Get Admin Result Format Example :

[
    {
        "adminID": 0,
        "username": "example_username",
        "password": "example_password",
    } ...
]

Create an Admin

To create a new administrator you need to make a POST request to the following url :
http://localhost/data_src/api/admin/create.php 



Create Admin JSON format Example :

{

    "adminID": int,
    "username": "string",
    "password": "string",
}

QUERY PARAMETERS

Field Type Description
username String Your username
password String Your password

Hangman

These are words used for the hangman word bank.

Get Word List

To get word list you need to make a GET call to the following url :
http://localhost/data_src/api/hangman/list.php 


Get Word List Result Format Example :
  
  [
      {
          "wordID": 1,
          "word": "Esbenshade",
      } ...
  ]

Get a Single Word

To get a specific product you need to make a GET call to the following url :
http://localhost/data_src/api/hangman/word.php 

The URL should look like:
  
  
  http://localhost/data_src/api/hangman/word.php
  
              
  Result if word exists:
  
  [
    {
      "word": "Esbenshade",
    }
  ]

Create a Word

To create a new word you need to make a POST request to the following url :
http://localhost/data_src/api/hangman/create.php 



  Create Word JSON format Example :
  
  {
  
      "wordID": int,
      "word": "string",
  }

QUERY PARAMETERS

Field Type Description
word String Your hangman word to be added

Delete a Word

To delete a word you need to make a DELETE request to the following url :
http://localhost/data_src/api/hangman/delete.php 



Delete Word JSON format Example :
{
  "word": "Esbenshade",
}

QUERY PARAMETERS

Field Type Description
word String Your hangman word to be deleted

Get Question List

To get question list you need to make a GET call to the following url :
http://localhost/data_src/api/question/read.php 


Get Question List Result Format Example :
  
  [
      {
          "questionID": 1,
          "question": "What year was Etown founded?",
      } ...
  ]

Get Single Question

To get a specific product you need to make a GET call to the following url :
http://localhost/data_src/api/question/questions.php 

The URL should look like:
  
  
  http://localhost/data_src/api/question/questions.php
  
              
  Result if question exists:
  
  [
    {
      "question": "What year was Etown founded?",
    }
  ]

Create a Question

To create a new question you need to make a POST request to the following url :
http://localhost/data_src/api/question/create.php 



  Create Question JSON format Example :
  
  {
  
      "questionID": int,
      "question": "string",
  }

QUERY PARAMETERS

Field Type Description
question String Your trivia question to be added

Delete a Question

To delete a word you need to make a DELETE request to the following url :
http://localhost/data_src/api/question/delete.php 



Delete Word JSON format Example :
{
  "questionID": 1,
}

QUERY PARAMETERS

Field Type Description
questionID Integer Your trivia question ID to be deleted

Users

These are the users of the website.

Create a User

To create a new user you need to make a POST request to the following url :
http://localhost/data_src/api/user/create.php 



Create User JSON format Example :

{

    "firstname": "string",
    "username": "string",
    "email": "string",
}

QUERY PARAMETERS

Field Type Description
firstname String The user's first name
username String The username of the user
email String The email of the user

Get User List

To get user list you need to make a GET call to the following url :
http://localhost/data_src/api/user/read.php 


Get User List Result Format Example :

[
{
  "firstname": "example_name",
  "username": "example_username",
  "email": "example_email",
} ...
]
